 # Prompt Engineering Assignment

This project analyzes the system prompt used by an AI voice agent for education loan collection calls.

## Approach

1. Reviewed the system prompt and identified design weaknesses.
2. Analyzed call transcripts to observe agent behavior.
3. Built a Python evaluation script to detect problematic responses.
4. Proposed improvements to the system prompt.

## Files

analysis/analyze_calls.py  
Script that processes transcripts and extracts agent responses.

analysis/prompt_evaluator.py  
Rule-based evaluator for identifying problematic agent responses.

analysis_results.json  
Output of the transcript evaluation.

prompt_autopsy_report.md  
Detailed analysis of prompt design flaws.

improved_system_prompt.md  
Improved version of the system prompt.
